# Assets Folder

This folder contains all static assets used in the portfolio.

## Structure

- **images/** - Project thumbnails, hero images, screenshots, and other image assets
- **icons/** - SVG icons, favicon, and icon assets
- **videos/** - Video files and video assets (if used)

## Usage

Reference assets in your code like:
```jsx
<img src="/assets/images/hero-image.jpg" alt="description" />
<img src="/assets/icons/my-icon.svg" alt="description" />
```
